# JAMB Mastermind 3000+

Complete JAMB Preparation Platform with 3000+ Questions

## Features
- 500+ questions per subject (8 subjects total)
- Full JAMB mock exams (180 questions, 2 hours)
- Performance analytics
- Offline functionality
- Progressive Web App (PWA) support

## Deployment Instructions
1. Upload all files to your web server
2. Ensure HTTPS is enabled (required for PWA)
3. Update contact information in `CONTACT.md`
4. Customize colors in CSS if needed

## Technical Requirements
- Modern web browser
- HTTPS for PWA features
- 100MB storage for icons and assets
